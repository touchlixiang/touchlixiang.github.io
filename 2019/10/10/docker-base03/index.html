04 Docker 容器管理

发表于 2019-10-10 | 分类于 Docker


启动容器 docker run

  1. docker 的容器是镜像的一个运行实例。
  2. docker 镜像是只读文件，而容器则带有运行时的可读写层，而且容器中的应用进程处于运行状态。
  • docker run ：创建一个新的容器并运行一个命令

进入容器

  • 进入容器的三种方法:attach(不推荐)、exec、

attach

  1. 进入到存活的容器中
  2. 如果再开一个窗口进来，操作是同步的，就像远程控制，单用户模式
  3. exit 就会推出, 生产不会使用这个命令

exec

  1. 对运行的容器执行指定命令
  2. 不想进入,让容器执行命令并返回结果

退出并不停止 docker

  1. crtl 先p 再q

nsenter

  1. nsenter 生产使用
  2. 即使exit 也不会关闭容器

查看日志 docker logs

copy容器文件 docker cp

  • 该命令支持在容器和主机之间复制文件。
